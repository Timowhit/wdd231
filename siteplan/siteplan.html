Site Plan: Magna Steyr Mesa Production Dashboard

1. Site Name

Mesa Production Dashboard (or WAML Dashboard)

Reason: Clearly identifies the location (Mesa, AZ facility) and primary function (production monitoring). Short and memorable for internal use.

2. Site Purpose

This internal web application provides real-time visibility into automotive production lines at the Magna Steyr Mesa facility. It displays current vehicle status, cycle times, overcycle alerts, First Time Complete percentage, and key performance indicators to help supervisors and managers quickly identify bottlenecks and maintain throughput targets.

3. Scenarios

  • As a line supervisor, I want to see which vehicle is currently on each line and how long it has been there so I can spot overcycle situations immediately.
  • As a plant manager, I want to view the overall FTC percentage and total overcycle time to assess shift performance at a glance.

4. Color Scheme

  • --lime (#a6b84b) → Good / on-time status, success
  • --red (#d72828) → Overcycle / warning / bad status
  • --amber (#e48c2a) → Attention / current vehicle highlight
  • #212529 dark gray → Main background
  • #2e73c7 blue → Accents, headers (if used)

These colors are already applied throughout this plan document.

5. Typography

Primary font: Arial, sans-serif (system default, highly readable in industrial environments)

  • Headings / line labels: bold, larger sizes (clamp-based)
  • Body / cell content: normal weight, 1.5vw+
  • KPIs: very large & bold for at-a-glance reading
